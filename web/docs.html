API Documentation

OpenAI-compatible inference API powered by your GPU.

Quick Start

Point any OpenAI-compatible client at your BareMetalRT instance. Replace the API key with one generated in Account Settings.

# Chat completion
curl https://baremetalrt.ai/v1/chat/completions \
  -H "Authorization: Bearer bmrt_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "phi-3-mini",
    "messages": [{"role": "user", "content": "Hello!"}],
    "max_tokens": 256
  }'

Authentication

All API requests require a Bearer token. Generate API keys in Account Settings.

Authorization: Bearer bmrt_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Endpoints

Chat Completions Live

POST /v1/chat/completions

Generate a chat response. Compatible with OpenAI's chat completions format.

ParameterTypeDescription
modelstringModel ID (e.g. "phi-3-mini")
messagesarrayArray of {role, content} message objects
max_tokensintegerMaximum tokens to generate (default: 256)
temperaturefloatSampling temperature 0-1 (default: 0.7)
streambooleanStream response as SSE (default: false)

List Models Live

GET /v1/models

List available models on your GPU.

Chat (SSE Streaming) Live

POST /api/chat

Native BareMetalRT chat endpoint with Server-Sent Events streaming. Used by the web app.

# Streaming chat
curl -X POST https://baremetalrt.ai/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!", "max_tokens": 128}'

GPU Status Live

GET /api/gpu-status

Check if a GPU daemon is connected.

# Response
{"connected": true}

Client Integration

Python (OpenAI SDK)

# pip install openai
from openai import OpenAI

client = OpenAI(
    base_url="https://baremetalrt.ai/v1",
    api_key="bmrt_your_api_key",
)

response = client.chat.completions.create(
    model="phi-3-mini",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(response.choices[0].message.content)

Cursor / Continue

In your IDE settings, set the API base URL to https://baremetalrt.ai/v1 and use your BareMetalRT API key.

JavaScript

const response = await fetch('https://baremetalrt.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer bmrt_your_api_key',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    model: 'phi-3-mini',
    messages: [{ role: 'user', content: 'Hello!' }],
  }),
});
const data = await response.json();
console.log(data.choices[0].message.content);
